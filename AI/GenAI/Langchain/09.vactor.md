# Vector Databases in LangChain 1.x (2026 Complete Guide)

# Overview

Vector Databases are used to store:

* Embeddings
* Documents
* Metadata
* Similarity Indexes

They are the foundation of:

* RAG Systems
* AI Agents
* Knowledge Bases
* Chatbots
* Search Systems

---

# What is a Vector Store?

Vector Store is LangChain's interface layer.

Example:

```python
vector_store.similarity_search(
    "Who is a bowler?"
)
```

Common Operations:

```python
add_documents()
similarity_search()
similarity_search_with_score()
delete()
update_document()
as_retriever()
```

# Chroma

## Installation

```bash
pip install langchain-chroma chromadb
```

## Import

```python
from langchain_chroma import Chroma
```

## Best For

* Learning
* Tutorials
* Local RAG
* Personal Projects

---

## Sample Code

```python
vector_store = Chroma(
    collection_name="sample",
    persist_directory="./db",
    embedding_function=OpenAIEmbeddings()
)

ids = vector_store.add_documents(docs)

results = vector_store.similarity_search(
    "Who is a bowler?"
)
```

---

## Output

```text
Jasprit Bumrah is considered one of the best fast bowlers...
```

---

## Pros

* Easy
* Beginner Friendly
* Local Storage

## Cons

* Not ideal for massive scale

Rating: 8/10

# FAISS

## Installation

```bash
pip install faiss-cpu
```

## Import

```python
from langchain_community.vectorstores import FAISS
```

## Best For

* Learning Similarity Search
* Fast Local Search
* Research

---

## Sample Code

```python
vector_store = FAISS.from_documents(
    docs,
    OpenAIEmbeddings()
)

results = vector_store.similarity_search(
    "Who is a bowler?"
)
```

---

## Output

```text
Jasprit Bumrah is a fast bowler
```

---

## Pros

* Extremely Fast
* Lightweight

## Cons

* No real database
* No persistence by default

Rating: 8.5/10

# Qdrant

## Installation

```bash
pip install langchain-qdrant qdrant-client
```

## Import

```python
from langchain_qdrant import QdrantVectorStore
```

## Best For

* AI Agents
* SaaS Products
* Production RAG

---

## Sample Code

```python
client = QdrantClient(":memory:")

vector_store = QdrantVectorStore.from_documents(
    docs,
    embedding=OpenAIEmbeddings(),
    client=client,
    collection_name="players"
)

results = vector_store.similarity_search(
    "Who is a bowler?"
)
```

---

## Output

```text
Jasprit Bumrah is considered one of the best fast bowlers...
```

---

## Pros

* Fast
* Open Source

```python
from dotenv import load_dotenv
from langchain_core.documents import Document
from langchain_chroma import Chroma
from langchain_openai import OpenAIEmbeddings

load_dotenv()

print("\n" + "=" * 80)
print("STEP 1: CREATE DOCUMENTS")
print("=" * 80)

docs = [
    Document(
        page_content="Virat Kohli is one of the most successful and consistent batsmen in IPL history.",
        metadata={"team": "Royal Challengers Bangalore"}
    ),
    Document(
        page_content="Rohit Sharma is the most successful captain in IPL history.",
        metadata={"team": "Mumbai Indians"}
    ),
    Document(
        page_content="MS Dhoni has led Chennai Super Kings to multiple IPL titles.",
        metadata={"team": "Chennai Super Kings"}
    ),
    Document(
        page_content="Jasprit Bumrah is considered one of the best fast bowlers in T20 cricket.",
        metadata={"team": "Mumbai Indians"}
    ),
    Document(
        page_content="Ravindra Jadeja is a dynamic all-rounder who contributes with both bat and ball.",
        metadata={"team": "Chennai Super Kings"}
    )
]

for i, doc in enumerate(docs, start=1):
    print(f"\nDocument {i}")
    print("Content :", doc.page_content)
    print("Metadata:", doc.metadata)

print("\n" + "=" * 80)
print("STEP 2: CREATE VECTOR STORE")
print("=" * 80)

vector_store = Chroma(
    collection_name="sample",
    persist_directory="./7.vector/my_chroma_db",
    embedding_function=OpenAIEmbeddings()
)

print("Vector Store Created Successfully")

print("\n" + "=" * 80)
print("STEP 3: INSERT DOCUMENTS")
print("=" * 80)

ids = vector_store.add_documents(docs)

print("Generated IDs:")
for idx, doc_id in enumerate(ids, start=1):
    print(f"Document {idx}: {doc_id}")

print("\n" + "=" * 80)
print("STEP 4: VIEW ALL STORED DOCUMENTS")
print("=" * 80)

data = vector_store.get(
    include=["documents", "metadatas"]
)

for i in range(len(data["documents"])):
    print(f"\nStored Document {i+1}")
    print("-" * 50)
    print("Content:")
    print(data["documents"][i])
    print("\nMetadata:")
    print(data["metadatas"][i])

print("\n" + "=" * 80)
print("STEP 5: SIMILARITY SEARCH")
print("=" * 80)

query = "Who among these are bowlers?"

print("Query:")
print(query)

results = vector_store.similarity_search(
    query=query,
    k=2
)

for i, doc in enumerate(results, start=1):
    print(f"\nMatch {i}")
    print("-" * 50)
    print(doc.page_content)
    print(doc.metadata)

print("\n" + "=" * 80)
print("STEP 6: SIMILARITY SEARCH WITH SCORE")
print("=" * 80)

results = vector_store.similarity_search_with_score(
    query=query,
    k=3
)

for i, (doc, score) in enumerate(results, start=1):
    print(f"\nMatch {i}")
    print("-" * 50)
    print("Score:", score)
    print("Content:", doc.page_content)
    print("Metadata:", doc.metadata)

print("\n" + "=" * 80)
print("STEP 7: METADATA FILTERING")
print("=" * 80)

filtered_docs = vector_store.similarity_search(
    query="",
    k=10,
    filter={"team": "Chennai Super Kings"}
)

for i, doc in enumerate(filtered_docs, start=1):
    print(f"\nCSK Player {i}")
    print(doc.page_content)
    print(doc.metadata)

print("\n" + "=" * 80)
print("STEP 8: UPDATE DOCUMENT")
print("=" * 80)

updated_doc = Document(
    page_content="""
    Virat Kohli is the highest run scorer in IPL history.
    Former RCB captain and one of the greatest T20 batsmen ever.
    """,
    metadata={"team": "Royal Challengers Bangalore"}
)

print("Updating Document ID:")
print(ids[0])

vector_store.update_document(
    document_id=ids[0],
    document=updated_doc
)

print("Document Updated Successfully")

print("\n" + "=" * 80)
print("STEP 9: VERIFY UPDATE")
print("=" * 80)

updated_data = vector_store.get(
    ids=[ids[0]],
    include=["documents", "metadatas"]
)

print(updated_data)

print("\n" + "=" * 80)
print("STEP 10: DELETE DOCUMENT")
print("=" * 80)

print("Deleting ID:")
print(ids[0])

vector_store.delete(ids=[ids[0]])

print("Document Deleted Successfully")

print("\n" + "=" * 80)
print("STEP 11: FINAL DATABASE STATE")
print("=" * 80)

final_data = vector_store.get(
    include=["documents", "metadatas"]
)

for i in range(len(final_data["documents"])):
    print(f"\nRemaining Document {i+1}")
    print(final_data["documents"][i])
    print(final_data["metadatas"][i])

print("\n")
print("=" * 80)
print("DEMO COMPLETED SUCCESSFULLY")
print("=" * 80)
```
