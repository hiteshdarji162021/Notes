## 1️⃣ Dynamic & Reusable Prompts

### ✅ Latest LangChain Code (2026)

```python
from langchain_core.prompts import PromptTemplate

prompt = PromptTemplate.from_template(
    "Summarize {topic} in {tone} tone"
)

result = prompt.invoke({
    "topic": "Cricket",
    "tone": "fun"
})

print(result)
```

### 💡 Why Useful?
- Reusable

## 3️⃣ Few-Shot Prompting

### ✅ Latest Code

```python
from langchain_core.prompts import FewShotPromptTemplate, PromptTemplate

examples = [
    {"input": "I was charged twice", "output": "Billing Issue"},
    {"input": "App crashes on login", "output": "Technical Problem"}
]

example_prompt = PromptTemplate.from_template(
    "Input: {input}\nCategory: {output}"
)

few_shot_prompt = FewShotPromptTemplate(
    examples=examples,
    example_prompt=example_prompt,
    prefix="Classify the issue",
    suffix="Input: {user_input}\nCategory:",
    input_variables=["user_input"]
)

result = few_shot_prompt.invoke({
    "user_input": "Payment failed but money deducted"
})

print(result)
```

### 💡 Why Useful?
- Improves accuracy
- Gives pattern to model
- Less hallucination

## ⚙️ Full Example (Prompt + LLM)

```python
from langchain_openai import ChatOpenAI
from langchain_core.prompts import ChatPromptTemplate

llm = ChatOpenAI(model="gpt-4o")

prompt = ChatPromptTemplate.from_messages([
    ("system", "You are a QA expert"),
    ("user", "Generate test cases for {feature}")
])

chain = prompt | llm

response = chain.invoke({
    "feature": "Login with OTP"
})

print(response.content)
```
