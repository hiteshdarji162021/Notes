# LangChain Document Loaders Canvas

## Objective

Load data from different sources and convert it into LangChain `Document` objects.

### Output Format

```python
Document(
    page_content="Actual Content",
    metadata={"source":"file/path"}
)
```

# 1. CSV Loader

## Purpose

Load data from CSV files.

### Code

```python
from langchain_community.document_loaders import CSVLoader

loader = CSVLoader(
    file_path='./5.Document_Loader/Social_Network_Ads.csv'
)

docs = loader.load()

print(len(docs))
print(docs[1])
```

### Flow

```text
CSV File
   ↓
CSVLoader
   ↓
Document Objects
   ↓
docs[]
```

# 3. Directory Loader

## Purpose

Load multiple PDF files from a folder.

### Code

```python
from langchain_community.document_loaders import DirectoryLoader, PyPDFLoader

loader = DirectoryLoader(
    path='./5.Document_Loader/books',
    glob='*.pdf',
    loader_cls=PyPDFLoader
)

# docs = loader.load()
docs = loader.lazy_load()

for document in docs:
    print(document.metadata)
```

### Flow

```text
Books Folder
    ↓
DirectoryLoader
    ↓
PDF Loader
    ↓
Document Stream
```

### load() vs lazy_load()

| load()                          | lazy_load()                |
| ------------------------------- | -------------------------- |
| Loads all documents into memory | Loads documents one by one |
| Better for small files          | Better for large files     |
| Higher memory usage             | Lower memory usage         |

# 4. Text Loader

## Purpose

Load plain text files.

### Code

```python
from langchain_community.document_loaders import TextLoader
from langchain_openai import ChatOpenAI
from langchain_core.output_parsers import StrOutputParser
from langchain_core.prompts import PromptTemplate
from dotenv import load_dotenv

load_dotenv()

model = ChatOpenAI()

prompt = PromptTemplate(
    template='write a summary for the following poem - \n {poem}',
    input_variables=['poem']
)

parser = StrOutputParser()

loader = TextLoader(
    './5.Document_Loader/cricket.txt',
    encoding='utf-8'
)

docs = loader.load()

chain = prompt | model | parser

print(chain.invoke({
    'poem': docs[0].page_content
}))
```

### Flow

```text
TXT File
   ↓
TextLoader
   ↓
Document[]
   ↓
Prompt
   ↓
LLM
   ↓
Summary
```

### Metadata Example

```python
{
    'source': 'cricket.txt'
}
```

### Use Cases

| Notes         | ⭐⭐⭐⭐   |

# 5. Web Loader

## Purpose

Load content directly from websites.

### Code

```python
from langchain_community.document_loaders import WebBaseLoader
from langchain_openai import ChatOpenAI
from langchain_core.output_parsers import StrOutputParser
from langchain_core.prompts import PromptTemplate
from dotenv import load_dotenv

load_dotenv()

model = ChatOpenAI()

prompt = PromptTemplate(
    template='''
    Answer the following question:
    {question}

    From the following text:
    {text}
    ''',
    input_variables=['question', 'text']
)

parser = StrOutputParser()

url = "https://www.flipkart.com/apple-macbook-air-m2-16-gb-256-gb-ssd-macos-sequoia-mc7x4hn-a/p/itmdc5308fa78421"

loader = WebBaseLoader(url)

docs = loader.load()

chain = prompt | model | parser

print(
    chain.invoke({
        'question': 'What is the product that we are talking about?',
        'text': docs[0].page_content
    })
)
```

### Flow

```text
Website URL
     ↓
WebBaseLoader
     ↓
HTML Fetch
     ↓
Text Extraction
     ↓
Document[]
     ↓
LLM
     ↓
Answer
```
