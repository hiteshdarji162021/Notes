# Agentic AI Canvas

## 1. What is Agentic AI?

Agentic AI is an AI system that: - Accepts a goal from a user - Plans
how to achieve it - Executes tasks autonomously - Adapts to changing
conditions - Requests human help only when necessary

### Formula

`Agentic AI = Goal + Planning + Reasoning + Tools + Memory + Adaptation + Autonomous Execution`

------------------------------------------------------------------------

## 2. Generative AI vs Agentic AI

  Generative AI        Agentic AI
  -------------------- ----------------------
  Reactive             Proactive
  Answers questions    Achieves goals
  Single response      Multi-step execution
  Human-driven         Autonomous
  Limited tool usage   Extensive tool usage

------------------------------------------------------------------------

## 4. Key Characteristics

### 1. Autonomy

-   Makes decisions independently
-   Executes actions without step-by-step instructions
-   Uses tools automatically

### 2. Goal-Oriented

-   Focuses on achieving a specific objective
-   Maintains progress until completion

### 3. Planning

-   Breaks large goals into smaller tasks
-   Generates multiple plans
-   Selects the best plan

### 4. Reasoning

-   Understands information
-   Draws conclusions
-   Makes intelligent decisions

### 5. Adaptability

-   Handles failures
-   Changes strategy when needed
-   Reacts to environmental feedback

### 6. Context Awareness

-   Remembers goals and constraints
-   Tracks progress
-   Uses previous information effectively

------------------------------------------------------------------------

## 5. Planning Lifecycle

### Step 1: Generate Candidate Plans

Multiple possible approaches are created.

### Step 2: Evaluate Plans

Evaluation criteria: - Efficiency - Cost - Risk - Tool availability -
Constraint alignment

### Step 3: Select Best Plan

Choose: - Human-approved plan - Policy-based plan - Highest scoring plan

------------------------------------------------------------------------

## 6. Reasoning Usage

-   Risk assessment

### During Execution

-   Decision making
-   Error handling
-   Human-in-the-loop decisions
-   Dynamic adjustments

## 7. Adaptability Triggers

### Tool Failure

Example: - Calendar API unavailable - Agent asks user directly

### External Feedback

Example: - Few job applicants - Modify JD or run ads

### Goal Change

Example: - Backend Engineer → Freelancer

------------------------------------------------------------------------

## 8. Autonomy Control Mechanisms

### Permission Scope

Define what actions are allowed.

### Human-in-the-Loop

Require approval before critical actions.

### Override Controls

Pause, stop, or modify behavior anytime.

### Guardrails & Policies

Define hard business and ethical rules.
