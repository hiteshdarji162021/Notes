# 📘 Python Set & Dictionary – Classroom Lesson

## 👩‍🏫 Teacher Mode (Simple English)

---

## 🧩 1. What is a Set?
A **set** is a collection of **unique items**.
- No duplicates allowed
- Unordered

### ✅ Example:
```python
nums = {1, 2, 3, 3}
print(nums)
```
👉 Output: {1, 2, 3}

---

## 🔧 Set Operations (Syntax → Code)

### ➕ Add Item
**Syntax:** set.add(value)
```python
nums.add(4)
print(nums)
```

---

### ❌ Remove Item
**Syntax:** set.remove(value)
```python
nums.remove(2)
print(nums)
```

---

### 🔄 Loop in Set
```python
for i in nums:
    print(i)
```

---

## 🧠 Use Case of Set
- Remove duplicate values
- Store unique users

```python
users = ["a", "b", "a", "c"]
unique_users = set(users)
print(unique_users)
```

---

## 🧩 2. What is a Dictionary?
A **dictionary** stores data in **key : value pair**.

### ✅ Example:
```python
student = {"name": "Hitesh", "age": 25}
print(student)
```

---

## 🔧 Dictionary Operations (Syntax → Code)

### ➕ Add / Update
**Syntax:** dict[key] = value
```python
student["city"] = "Ahmedabad"
student["age"] = 26
print(student)
```

---

### 🔍 Access Value
**Syntax:** dict[key]
```python
print(student["name"])
```

---

### ❌ Delete Item
**Syntax:** del dict[key]
```python
del student["age"]
print(student)
```

---

### 🔄 Loop Dictionary
```python
for key, value in student.items():
    print(key, value)
```

---

## ⚖️ Set vs Dictionary (Comparison Table)

| Feature | Set | Dictionary | Rating |
|--------|-----|------------|--------|
| Structure | values only | key:value | ⭐⭐⭐⭐⭐ |
| Duplicate | Not allowed | Keys not duplicate | ⭐⭐⭐⭐⭐ |
| Order | Unordered | Ordered (Python 3.7+) | ⭐⭐⭐⭐ |
| Use | Unique data | Structured data | ⭐⭐⭐⭐⭐ |

---

## 🧠 When to Use (Practical Use Cases)

### ✅ Use Set when:
```python
emails = ["a@gmail.com", "b@gmail.com", "a@gmail.com"]
print(set(emails))
```
👉 Use case: remove duplicates, unique entries

---

### ✅ Use Dictionary when:
```python
user = {"id": 1, "name": "Hitesh"}
```
👉 Use case: API data, user info, config data
