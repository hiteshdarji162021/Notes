## ⚖️ 1. Comparison Operators (True/False)

Used to compare values.

### 🔹 Operators Table

| Operator | Meaning          | Example | Result |
|----------|------------------|--------|--------|
| ==       | Equal to         | 5 == 5 | True   |
| !=       | Not equal        | 5 != 3 | True   |
| >        | Greater than     | 10 > 5 | True   |
| <        | Less than        | 10 < 5 | False  |
| >=       | Greater or equal | 5 >= 5 | True   |
| <=       | Less or equal    | 4 <= 5 | True   |

### 🔹 Real-Life Example

```python
age = 20

if age >= 18:
    print("Allowed to vote")
else:
    print("Not allowed")
```

👉 Used in:
- Login validation
- Eligibility check
- Test automation assertions

## 🔗 2. Logical Operators (Combine Conditions)

### 🔹 Operators Table

| Operator | Meaning        | Example        | Result |
|----------|----------------|--------------|--------|
| and      | Both true      | True and False | False |
| or       | Any one true   | True or False | True  |
| not      | Reverse result | not True      | False |

### 🔹 Real-Life Example

```python
age = 25
has_id = True

if age > 18 and has_id:
    print("Entry allowed")
```

👉 Used in:
- Security systems
- Multi-condition checks

## 🧩 3. Assignment Operators

Used to assign values to variables.

| Operator | Example  | Meaning        |
|----------|----------|----------------|
| =        | x = 5    | Assign value   |
| +=       | x += 3   | x = x + 3      |
| -=       | x -= 2   | x = x - 2      |
| *=       | x *= 2   | x = x * 2      |
| /=       | x /= 2   | x = x / 2      |

### 🔹 Example

```python
x = 10
x += 5
print(x)  # 15
```

---

## 🧪 4. Membership Operators

Used to check if value exists in a sequence.

| Operator | Example        | Result |
|----------|---------------|--------|
| in       | 'a' in 'cat'  | True   |
| not in   | 'x' in 'cat'  | False  |

### 🔹 Example

```python
text = "python"

print("p" in text)   # True
print("z" not in text) # True
```

---

## 🆔 5. Identity Operators

Used to compare memory location.

| Operator | Example     | Result |
|----------|------------|--------|
| is       | x is y     | True/False |
| is not   | x is not y | True/False |

### 🔹 Example

```python
a = [1, 2]
b = a

print(a is b)  # True (same object)
```
