# 🧠 Logical Operators Practice (Student Questions)

This worksheet is designed for students to practice logical operators (`and`, `or`, `not`) with real-life scenarios.

---

## 🥇 Program 1: Job Eligibility Checker

### 📌 Question:
Write a Python program to check whether a person is eligible for a job.

### 📝 Requirements:
1. Take age as input from the user.
2. Ask the user if they have a degree (Yes/No).
3. Check if BOTH conditions are true:
   - Age should be 21 or above
   - User must have a degree
4. If both conditions are true, print:
   👉 "Eligible for job"
5. Otherwise, print:
   👉 "Not eligible for job"

---

## 🥈 Program 2: Secure Login System

### 📌 Question:
Write a Python program to simulate a login system.

### 📝 Requirements:
1. Take username as input.
2. Take password as input.
3. Assume a variable `is_blocked = True`.
4. Check the following conditions:
   - Username should match "admin"
   - Password should match "1234"
   - User should NOT be blocked (`is_blocked` should not be True)
5. If all conditions are satisfied, print:
   👉 "Login Successful"
6. Otherwise, print:
   👉 "Access Denied"

---

# 🚀 Instructions for Students

- Use `if-else` conditions properly
- Use logical operators (`and`, `not`)
- Try to test with different inputs
