# 📘 MODULE 1: BASICS (Quick Revision)

## 🔹 1. Python Introduction
- Python is a **simple and powerful programming language**
- Used in:
  - Web Development
  - AI / Machine Learning
  - Automation (QA, Testing)
  - Data Analysis

---

## 🔹 2. Print Statement

### Single Line
```python
print("Hello World")
```

### Multiple Line
```python
print("Hello")
print("Python")
```

---

## 🔹 3. Comments

### Single Line
```python
# This is a comment
```

### Multi-line
```python
"""
This is a multi-line comment
"""
```

👉 Comments are used for explanation and ignored by Python.

---

## 🔹 4. Variables & Data Types

```python
name = "Hitesh"   # string
age = 25          # integer
price = 99.5      # float
is_active = True  # boolean
```

👉 Variables store values
👉 No need to declare type in Python

---

## 🔹 5. Taking Input

```python
name = input("Enter your name: ")
print(name)
```

👉 Input is always taken as **string by default**

---

## 🔹 6. Type Casting

```python
age = int(input("Enter age: "))
price = float(input("Enter price: "))
num = str(100)
```

👉 Used to convert one data type to another

---

# ➕ MODULE 2: OPERATORS & CONDITIONS (Quick Revision)

## 🔹 7. Arithmetic Operators

```python
print(10 + 5)   # Addition
print(10 - 5)   # Subtraction
print(10 * 5)   # Multiplication
print(10 / 5)   # Division
print(10 % 3)   # Modulus
print(10 // 3)  # Floor Division
print(2 ** 3)   # Power
```

---

## 🔹 8. Assignment Operators

```python
x = 10
x += 5   # 15
x -= 3   # 12
x *= 2   # 24
x /= 4   # 6
```

---

## 🔹 9. Simple if-else

```python
age = 18

if age >= 18:
    print("Adult")
else:
    print("Minor")
```

👉 Used for decision making

---

✔ Use type casting carefully  
✔ Use operators for calculations  
✔ Use if-else for decisions
