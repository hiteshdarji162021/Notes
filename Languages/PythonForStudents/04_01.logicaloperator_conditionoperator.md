# 🧠 Python Conditional Statements (Teacher Guide)

---

## 📘 1. What is a Condition?

A condition is a **True or False check**.

👉 Python uses conditions to take decisions.

Example:
```python
age = 20
print(age >= 18)  # True
```

---

# ⚖️ 2. Comparison Operators (True/False)

Used to compare values.

| Operator | Meaning          | Example  | Result |
| -------- | ---------------- | -------- | ------ |
| ==       | Equal to         | 5 == 5   |        |
| !=       | Not equal        | 5 != 3   |        |
| >        | Greater than     | 10 > 5   |        |
| >        | Less than        | 10 > 10  |        |
| >=       | Less than        | 10 >= 10 |        |
| <        | Less than        | 10 < 10  |        |
| >=       | Greater or equal | 5 <=10>  |        |
| <=       | Less or equal    | 4 <= 5   |        |

---

## 💻 Example

```python
age = 20

if age >= 18:
    print("Allowed to vote")
else:
    print("Not allowed")
```

👉 Used in:
- Login validation
- Eligibility check
- Test automation assertions

---

# 🔗 3. Logical Operators (Combine Conditions)

| Operator | Meaning      | Example        | Result |
| -------- | ------------ | -------------- | ------ |
| and      | Both true    | True and False | False  |
| or       | Any one true | True or False  | True   |
| not      | Reverse      | not True       | False  |

---

## 💻 Example

```python
username = input("Enter username: ")
password = input("Enter password: ")

if username == "admin" and password == "1234":
    print("Login Successful")
else:
    print("Invalid Credentials")
```

```python

is_blocked = True

if  not is_blocked:
    print("Login Allowed")

    print("Access Denied")
```


```python
age = int(input("Enter your age: "))

if not (age < 18):

```

👉 Used in:
- Security systems

# 🔀 4. if, elif, else

## 🔹 if
Runs when condition is True

```python
age = 18

if age >= 18:
    print("Adult")
```

---

## 🔹 if-else
Two outcomes

```python
age = 16

if age >= 18:
    print("Adult")
else:
    print("Minor")
```

---

## 🔹 if-elif-else
Multiple conditions

```python
marks = 75

if marks >= 90:
    print("Grade A")
elif marks >= 70:
    print("Grade B")
elif marks >= 50:
    print("Grade C")
else:
    print("Fail")
```

---

# 🧩 5. Nested if

👉 One condition inside another

```python
num = int(input("Enter a number: "))

if num >= 0:
    if num % 2 == 0:
        print("Positive Even Number")
    else:
        print("Positive Odd Number")
else:
    print("Negative Number")
```

# 🔁 6. match-case (Switch Alternative)

```python
day = 2

match day:
    case 1:
        print("Monday")
    case 2:
        print("Tuesday")
    case 3:
        print("Wednesday")
    case _:
        print("Invalid")
```

# 🧩 7. Default Case in match-case

In Python, `match-case` also supports a **default case**.

👉 It is written as:
```python
case _
```

👉 `_` means: "anything else"

### 💻 Example
```python
day = 'chrome'

match day:
    case 'chrome':
        print("chrome browser open")
    case 'firefox':
        print("firefox browser open")
    case _:
        print("Invalid day")
```

# ⚖️ 8. When to Use if-else vs match-case

## ✅ Use if-else when:
- Conditions are complex
- You use logical operators (and, or)
- You check ranges (>, <, >=)

### Example
```python
marks = 85

if marks >= 90:
    print("A")
elif marks >= 70:
    print("B")
else:
    print("C")
```

---

## ✅ Use match-case when:
- You compare exact values
- You have fixed options (menu, status)
- You want cleaner code for many cases

### Example
```python
status = 200

match status:
    case 200:
        print("Success")
    case 404:
        print("Not Found")
    case _:
        print("Error")
```

---

# 🎯 Final Rule

👉 Use **if-else** → for logic and conditions  
👉 Use **match-case** → for fixed values and options
