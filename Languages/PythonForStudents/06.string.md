# 📘 Python Strings – Classroom Lesson (For 12 Students)

## 👩‍🏫 Teacher Mode: Simple & Clear Explanation

---

## 🧩 1. What is a String?
A **string** is a collection of characters (letters, numbers, symbols).

### ✅ Example:
```python
name = "Hitesh"
city = "Ahmedabad"
```

---

## 🔗 2. String Concatenation (Joining Strings)
We use `+` to join strings.

### ✅ Example:
```python
first_name = "Hitesh"
last_name = "Darji"
full_name = first_name + " " + last_name
print(full_name)
```

### 🧠 Output:
```
Hitesh Darji
```

---

## 🔠 3. String Functions (Manipulation)

### 🔹 upper() → Convert to CAPITAL letters
```python
text = "hello"
print(text.upper())
```
👉 Output: HELLO

---

### 🔹 lower() → Convert to small letters
```python
text = "HELLO"
print(text.lower())
```
👉 Output: hello

---

### 🔹 replace() → Replace word or letter
```python
text = "I like tea"
print(text.replace("tea", "coffee"))
```
👉 Output: I like coffee

---

### 🔹 split() → Break string into list
```python
text = "apple mango banana"
print(text.split())
```
👉 Output: ['apple', 'mango', 'banana']

---

## 🧾 4. Printing with Quotes & Formatting

### 🔹 Single Quotes (' ')
```python
print('Hello Students')
```

### 🔹 Double Quotes (" ")
```python
print("Hello Students")
```

### 🔹 Triple Quotes (''' ''' or """ """) → Multi-line text
```python
print('''This is line 1
This is line 2
This is line 3''')
```

---

## 🔗 5. Concatenation using f-string and comma

### 🔹 Using f-string (Recommended 👍)
```python
name = "Hitesh"
age = 30
print(f"My name is {name} and I am {age} years old")
```

👉 Output:
```
My name is Hitesh and I am 30 years old
```

---

### 🔹 Using comma ( , ) in print
```python
name = "Hitesh"
age = 30
print("My name is", name, "and I am", age, "years old")
```

👉 Output:
```
My name is Hitesh and I am 30 years old
```

---

## 🧠 Teacher Note
- f-string is modern and clean
- comma automatically adds space between values
- use triple quotes f
