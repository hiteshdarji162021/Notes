# 📘 Python List & Tuple – Classroom Lesson

## 👩‍🏫 Teacher Mode (Simple English)

---

## 🧩 1. What is a List?
A **list** is a collection of items.
- It is **ordered**
- It is **changeable (mutable)**

### ✅ Example:
```python
fruits = ["apple", "banana", "mango"]
print(fruits)
```

---

## 🔧 List Operations

### 🔹 Access Item
```python
print(fruits[0])  # apple
```

### 🔹 Change Item
```python
fruits[1] = "orange"
print(fruits)
```

### 🔹 Add Item
```python
fruits.append("grapes")
```

### 🔹 Remove Item
```python
fruits.remove("apple")
```

---

## 🧠 2. What is a Tuple?
A **tuple** is also a collection of items.
- It is **ordered**
- It is **NOT changeable (immutable)**

### ✅ Example:
```python
colors = ("red", "green", "blue")
print(colors)
```

---

## 🔧 Tuple Operations

### 🔹 Access Item
```python
print(colors[1])  # green
```

### 🔹 Not Allowed ❌
```python
colors[1] = "yellow"  # Error
```

---

## 🔧 List CRUD Operations (Clear Syntax → Code Flow)

### 📌 Step 1: Create List
```python
nums = [1, 2, 3]
print(nums)
```

---

### ➕ Step 2: Add (Append / Insert)
**Syntax:**
- list.append(value)
- list.insert(index, value)

```python
nums.append(4)        # add at end
nums.insert(1, 10)    # add at index 1
print(nums)
```

---

### ✏️ Step 3: Update (Change Value)
**Syntax:**
- list[index] = new_value

```python
nums[0] = 100
print(nums)
```

---

### ❌ Step 4: Delete (Remove / Pop / del)
**Syntax:**
- list.remove(value)
- list.pop()
- del list[index]

```python
nums.remove(10)   # remove by value
nums.pop()        # remove last item
del nums[1]       # remove by index
print(nums)
```

---

## 🔧 Tuple Notes (No CRUD Allowed)
- Tuple is **immutable** → cannot Add / Update / Delete

### 📌 Example (Top to Bottom)
```python
t = (1, 2, 3)
print(t)

print(t[0])   # access allowed

# t[0] = 5    ❌ Error (not allowed)
# t.append(4) ❌ Error
```

---
- Tuple is **immutable** → cannot Add/Update/Delete
- You can only **read (access)** items
```python
t = (1, 2, 3)
print(t[0])  # 1
# t[0] = 5  ❌ Error
```

---

## ⚖️ List vs Tuple (CRUD Comparison Table)

| Operation | List Syntax | Tuple Syntax | Allowed in Tuple? | Rating |
|----------|------------|--------------|-------------------|--------|
| Create | nums = [1,2,3] | t = (1,2,3) | Yes | ⭐⭐⭐⭐⭐ |
| Add | nums.append(4) | Not Available | ❌ No | ⭐⭐⭐⭐⭐ |
| Insert | nums.insert(1,10) | Not Available | ❌ No | ⭐⭐⭐⭐ |
| Update | nums[0] = 100 | Not Available | ❌ No | ⭐⭐⭐⭐⭐ |
| Delete | nums.remove(2) / pop() / del | Not Available | ❌ No | ⭐⭐⭐⭐⭐ |
| Access | nums[0] | t[0] | Yes | ⭐⭐⭐⭐⭐ |

## 🧠 When to Use (Practical Use Cases)

### ✅ Use List when (Dynamic Data)
```python
cart = ["milk", "bread"]
cart.append("eggs")
cart[0] = "butter"
```
👉 Use case: shopping cart, test data, API response list

---

### ✅ Use Tu

### ✅ Use List when:
- Data **changes frequently**
- Example:
```python
shopping = ["milk", "bread"]
shopping.append("eggs")
```
👉 Real life: shopping cart, user inputs, test cases list

---

### ✅ Use Tuple when:
- Data should be **fixed / constant**
- Example:
```python
colors = ("red", "green", "blue")
```
👉 Real life: RGB values, coordinates (x,y), config values
