# 🔁 Python Loops – Complete Teacher Guide (Theory + Practical)

---

## 📘 1. What are Loops?
Loops are used to **repeat a block of code multiple times**.

👉 Instead of writing:
```
print(1)
print(2)
print(3)
```
We use loops to automate repetition.

---

## 🎯 Why use loops?
- Reduce code repetition
- Improve readability
- Handle large data efficiently
- Used in automation, testing, data processing

---

# 🔄 2. Types of Loops in Python

1. `while` loop → condition-based
2. `for` loop → iteration-based

---

# 🔁 3. while Loop

Runs **as long as condition is True**.

### 🔹 Syntax
```python
while condition:
    statements
    increment / decrement
```

### 💻 Example – Print 1 to 5
```python
i = 1
while i <= 5:
    print(i)
    i += 1
```

---

### ⚠️ Infinite Loop Example
```python
while True:
    print("This runs forever")
```
👉 Always use `break` to stop it.

---

# 🔢 4. range() Function

`range()` generates a **sequence of numbers**.

### 🔹 Syntax
```python
range(start, stop, step)
```

### 🔹 Examples
```
range(5)        → 0 1 2 3 4
range(1, 6)     → 1 2 3 4 5
range(1, 10, 2) → 1 3 5 7 9
```

---

# 🔂 5. for Loop

Used when number of iterations is **known**.

### 🔹 Syntax
```python
for variable in range(start, stop, step):
    statements
```

### 💻 Example – Print 1 to 5
```python
for i in range(1, 6):
    print(i)
```

---

### 🔹 Loop through List
```python
fruits = ["apple", "banana", "mango"]

for fruit in fruits:
    print(fruit)
```

---

### 🔹 Loop through String
```python
for ch in "python":
    print(ch)
```

---

# 🔁 6. Nested Loops

👉 Loop inside another loop

### 💻 Example – Pattern
```python
for i in range(1, 4):
    for j in range(1, 4):
        print(i, j)
```

---

# ⏭️ 7. Jumping Statements

## 🔹 break
Stops the loop immediately

```python
for i in range(1, 10):
    if i == 5:
        break
    print(i)
```

---

## 🔹 continue
Skips current iteration

```python
for i in range(1, 6):
    if i == 3:
        continue
    print(i)
```

---

## 🔹 pass
Does nothing (placeholder)

```python
for i in range(3):
    pass
```

---

# 🔚 8. else with Loop (Important)

👉 Runs when loop completes normally (no break)

```python
for i in range(3):
    print(i)
else:
    print("Loop finished")
```

---

# 🧠 9. Advanced Loop Concepts

## 🔹 enumerate()
Gives index + value

```python
names = ["A", "B", "C"]

for index, value in enumerate(names):
    print(index, value)
```

---

## 🔹 zip()
Loop multiple lists together

```python
names = ["A", "B"]
scores = [90, 80]

for n, s in zip(names, scores):
    print(n, s)
```

---

## 🔹 List Comprehension (Pythonic Loop)

```python
nums = [i for i in range(1, 6)]
print(nums)
```

# 💻 10. Practice Programs

## 1. Even or Odd
```python
num = 7

if num % 2 == 0:
    print("Even")
else:
    print("Odd")
```

---

## 2. Sum of Digits
```python
num = 123
sum = 0

while num > 0:
    sum += num % 10
    num //= 10

print(sum)
```

---

## 3. Reverse Number
```python
num = 123
rev = 0

while num > 0:
    rev = rev * 10 + num % 10
    num //= 10

print(rev)
```

---

## 4. Multiplication Table
```python
num = 5

for i in range(1, 11):
    print(num * i)
```

---

## 5. Count Vowels
```python
text = "python"
count = 0

for ch in text:
    if ch in "aeiou":
        count += 1

print(count)
```

# 🎯 11. Student Tips

✔ Use `while` → when condition-based loop needed  
✔ Use `for` → when iterations are fixed  
✔ Use `range()` → for numeric loops  
✔ Use `break` → stop loop  
✔ Use `continue` → skip iteration  
✔ Use `else` with loop for completion logic
