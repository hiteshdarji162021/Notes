## 🖨️ 1. Print (Single Line)

### Single Quote અને Double Quote
```python

print("Hello World")
```

👉 બંને same output આપે છે

---

## 🧾 2. Multi Line Print

### Method 1: Triple Quotes
```python
print("""Hello
This is Python
Multi-line text""")
```

### Method 2: \n (New Line)
```python
print("Hello\nThis is Python\nMulti-line text")
```

## 🧠 4. Variable Naming Rules

### Valid Rules
- Letter અથવા underscore થી શરૂ થવું જોઈએ
- Number થી શરૂ ન થવું જોઈએ
- Letters, numbers, underscore use કરી શકાય
- Case-sensitive છે
- Python keyword ન હોવું જોઈએ

---

### ✅ Valid Examples
```python
myvar = "John"
_myvar = "John"
my_var = "John"  (most prefrred)
myVar = "John"
myvar2 = "John"
```

### ❌ Invalid Examples
```python
2myvar = "John"   # number થી શરૂ
my-var = "John"  # hyphen not allowed
my var = "John"  # space not allowed
```
