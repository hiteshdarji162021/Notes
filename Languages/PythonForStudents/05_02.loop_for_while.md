## 📘  What are Loops?
Loops are used to **repeat a block of code multiple times**.

👉 Instead of writing:
```
print(1)
print(2)
print(3)
```
We use loops to automate repetition.

---

## 🎯 Why use loops?
- Reduce code repetition
- Improve readability
- Handle large data efficiently
- Used in automation, testing, data processing

---

# 🔄 Types of Loops in Python

1. `while` loop → condition-based
2. `for` loop → iteration-based

---

# 🔁 while Loop

Runs **as long as condition is True**.

### 🔹 Syntax
```python
while condition:
    statements
    increment / decrement
```

### 💻 Example – Print 1 to 5
```python
i = 1
while i <= 5:
    print(i)
    i += 1
```

---

### ⚠️ Infinite Loop Example
```python
while True:
    print("This runs forever")
```
👉 Always use `break` to stop it.

---

# 🔢 range() Function

`range()` generates a **sequence of numbers**.

### 🔹 Syntax
```python
range(start, stop, step)
```

### 🔹 Examples
```
range(5)        → 0 1 2 3 4
range(1, 6)     → 1 2 3 4 5
range(1, 10, 2) → 1 3 5 7 9
```

---

# 🔂 for Loop

Used when number of iterations is **known**.

### 🔹 Syntax
```python
for variable in range(start, stop, step):
    statements
```

### 💻 Example – Print 1 to 5
```python
for i in range(1, 6):
    print(i)
```

---

### 🔹 Loop through List
```python
fruits = ["apple", "banana", "mango"]

for fruit in fruits:
    print(fruit)
```

---

### 🔹 Loop through String
```python
for ch in "python":
    print(ch)
```

---

# 🔁 6. Nested Loops

👉 Loop inside another loop

### 💻 Example – Pattern
```python
for i in range(1, 4):
    for j in range(1, 4):
        print(i, j)
```

# ⏭️ 7. Jumping Statements

## 🔹 break
Stops the loop immediately

```python
for i in range(1, 10):
    if i == 5:
        break
    print(i)
```

---

## 🔹 continue
Skips current iteration

```python
for i in range(1, 6):
    if i == 3:
        continue
    print(i)
```

---

## 🔹 pass
Does nothing (placeholder)

```python
for i in range(3):
    pass
```

---

# 🔚 8. else with Loop (Important)

👉 Runs when loop completes normally (no break)

```python
for i in range(3):
    print(i)
else:
    print("Loop finished")
```


---

## 🔹 List Comprehension (Pythonic Loop)

```python
nums = [i for i in range(1, 6)]
print(nums)
```

# 🎯 11. Student Tips

✔ Use `while` → when condition-based loop needed  
✔ Use `for` → when iterations are fixed  
✔ Use `range()` → for numeric loops  
✔ Use `break` → stop loop  
✔ Use `continue` → skip iteration  
✔ Use `else` with loop for completion logic
