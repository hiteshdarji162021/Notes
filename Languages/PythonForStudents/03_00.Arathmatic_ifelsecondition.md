# 🧠 Python Arithmetic Operators & If-Else – Step-by-Step Guide

This guide is made for **absolute beginners**. Follow step by step and practice each part.

---

## 🎯 Step 1: What are Operators?

Operators are symbols that **perform operations on values**.

👉 Example:
```python
10 + 5
```
Here `+` adds two numbers.

---

## ➕ Step 2: Arithmetic Operators (Math Basics)

Used for **basic calculations**.

### 🔹 Common Operators

| Operator | Name           | Example | Output |
|----------|----------------|--------|--------|
| +        | Addition       | 10 + 5 | 15     |
| -        | Subtraction    | 10 - 5 | 5      |
| *        | Multiplication | 10 * 5 | 50     |
| /        | Division       | 10 / 5 | 2.0    |
| %        | Modulus        | 10 % 3 | 1      |
| **       | Power          | 2 ** 3 | 8      |
| //       | Floor Division | 10 // 3| 3      |

---

## 💻 Step 3: Try Simple Code

```python
a = 10
b = 3

print(a + b)   # 13
print(a - b)   # 7
print(a * b)   # 30
print(a / b)   # 3.33
print(a % b)   # 1
print(a ** b)  # 1000
print(a // b)  # 3
```

👉 Practice tip: Change values of `a` and `b` and observe output.

---

## 🛒 Step 4: Real-Life Example (Billing)

```python
price = 100
quantity = 3

bill = price * quantity
print("Total Bill:", bill)
```

👉 Output:
```
Total Bill: 300
```

---

## 🤔 Step 5: What are Conditional Statements?

Used to **make decisions in code**.

👉 Real life:
- If raining → take umbrella
- Else → don’t take umbrella

---

## ⚠️ Step 6: Important Rule (Indentation)

Python uses **spaces (indentation)** to define blocks.

❌ Wrong:
```python
if age > 18:
print("Adult")
```

✅ Correct:
```python
if age > 18:
    print("Adult")
```

👉 Always use **4 spaces**.

---

## ✅ Step 7: if Statement (Single Condition)

Runs code only when condition is **True**.

### Syntax
```python
if condition:
    statement
```

### Example
```python
age = 18

if age >= 18:
    print("You can vote")
```

---

## 🔀 Step 8: if...else (Two Conditions)

Used when there are **two outcomes**.

### Example
```python
age = 16

if age >= 18:
    print("Eligible")
else:
    print("Not Eligible")
```

---

## 🔁 Step 9: if...elif...else (Multiple Conditions)

Used when there are **many conditions**.

### Example
```python
marks = 60

if marks >= 90:
    print("Grade A")
elif marks >= 75:
    print("Grade B")
elif marks >= 50:
    print("Grade C")
else:
    print("Fail")
```

---

## 🔗 Step 10: Combine Operators + If-Else

```python
price = 500

if price > 300:
    discount = price * 0.1
    final_price = price - discount
    print("Discount Applied:", final_price)
else:
    print("No Discount")
```
