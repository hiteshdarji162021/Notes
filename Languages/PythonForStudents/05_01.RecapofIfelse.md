# 🧠 Python If Conditions – Quick Student Recap

Hello students 👨‍🏫,
This is a **quick revision guide** for everything we learned about **if conditions in Python**.

---

## ✅ 1. Simple if
👉 Executes block only when condition is TRUE

```python
age = 18

if age >= 18:
    print("You can vote")
```

---

## ✅ 2. if with else
👉 One block runs if TRUE, other runs if FALSE

```python
age = 16

if age >= 18:
    print("Eligible")
else:
    print("Not Eligible")
```

---

## ✅ 3. if with AND condition
👉 Both conditions must be TRUE

```python
username = input("Enter username: ")
password = input("Enter password: ")

if username == "admin" and password == "1234":
    print("Login Successful")
else:
    print("Invalid Credentials")
```

---

## ✅ 4. if with OR condition
👉 At least one condition must be TRUE

```python
username = input("Enter username: ")
password = input("Enter password: ")

if username == "admin" or password == "1234":
    print("Login Successful")
else:
    print("Invalid Credentials")
```

---

## ✅ 5. if with NOT condition
👉 Reverses the condition

```python
is_blocked = False

if not is_blocked:
    print("Login allowed")
```

---

## ✅ 6. if - elif - else
👉 Multiple conditions check one by one

```python
marks = 75

if marks >= 90:
    print("Grade A")
elif marks >= 60:
    print("Grade B")
else:
    print("Grade C")
```

---

## ✅ 7. Nested if-else
👉 if inside another if

```python
username = input("Enter username: ")
password = input("Enter password: ")

if username == "admin" :
    if password == "1234":
       print("Login Successful")
    else:
      print("Invalid passowrd")
else:
    print ('invalid username')
```

---

## ✅ 8. Check value in range
👉 Check if value lies between two numbers

```python
number = 15

if number>=10 and number<=20:
    print("Number is in range")
else:
    print("Out of range")
```

- Always use **indentation (spaces)** properly
- Conditions must return **True or False**
