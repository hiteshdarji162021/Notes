# 🧠 Python If Conditions – Quick Student Recap

Hello students 👨‍🏫,
This is a **quick revision guide** for everything we learned about **if conditions in Python**.

---

## ✅ 1. Simple if
👉 Executes block only when condition is TRUE

```python
age = 18

if age >= 18:
    print("You can vote")
```

---

## ✅ 2. if with else
👉 One block runs if TRUE, other runs if FALSE

```python
age = 16

if age >= 18:
    print("Eligible")
else:
    print("Not Eligible")
```

---

## ✅ 3. if with AND condition
👉 Both conditions must be TRUE

```python
age = 25
has_degree = True

if age >= 18 and has_degree:
    print("Eligible for job")
```

---

## ✅ 4. if with OR condition
👉 At least one condition must be TRUE

```python
marks = 35
sports_quota = True

if marks >= 40 or sports_quota:
    print("Pass")
```

---

## ✅ 5. if with NOT condition
👉 Reverses the condition

```python
is_blocked = False

if not is_blocked:
    print("Login allowed")
```

---

## ✅ 6. if - elif - else
👉 Multiple conditions check one by one

```python
marks = 75

if marks >= 90:
    print("Grade A")
elif marks >= 60:
    print("Grade B")
else:
    print("Grade C")
```

---

## ✅ 7. Nested if-else
👉 if inside another if

```python
age = 20
has_id = True

if age >= 18:
    if has_id:
        print("Entry allowed")
    else:
        print("ID required")
else:
    print("Not allowed")
```

---

## ✅ 8. Check value in range
👉 Check if value lies between two numbers

```python
number = 15

if 10 <= number <= 20:
    print("Number is in range")
else:
    print("Out of range")
```

---

# 🎯 Final Tip
- Always use **indentation (spaces)** properly
- Conditions must return **True or False**
- Practice daily with small examples
