### Concept

> A **callback** is a function passed as a parameter to another function.

## 3.1 Callback with Arrow Function

### Code

```js
let sum = (a, b) => a + b;

function calculator(callback, a, b) {
  return callback(a, b);
}

let val = calculator(sum, 20, 10);
console.log(val);
```

### Output

```
30
```

### Use Case

- Utility functions
- Calculations
- Automation helpers

---

## 3.2 Callback with Anonymous Function

### Code

```js
let multiplication = function (a, b) {
  return a * b;
};

function cal(a, b, callback) {
  return callback(a, b);
}

let result = cal(10, 20, multiplication);
console.log(result);
```

### Output

```
200
```

### Use Case

- One-time logic
- Cleaner APIs

## 3.3 Callback with Named Function

## 3.4 Inline Callback (Most Used in Industry)

### Code

```js
function doSomething(callback, url) {
  console.log("enter the url", url);
  callback();
}

doSomething(() => {
  console.log("hello app");
}, "https://www.google.com");
```

### Output

```
enter the url https://www.google.com
hello app
```

### Use Case

- Event handlers
- Async operations
- Cypress / Playwright commands

## 3.5 Multiple Callbacks


### Output

```

```

### Use Case

- Success & failure handlers
- Validation pipelines
