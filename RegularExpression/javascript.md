# JavaScript Installation Guide

## 1. Introduction
JavaScript is a programming language mainly used for web development. To start learning JavaScript, you need a browser and a code editor.

## 2. Install Node.js
Node.js allows JavaScript to run outside the browser.

Steps:
1. Go to https://nodejs.org
2. Download the LTS version.
3. Run the installer.
4. Follow the installation steps.

## 3. Verify Installation
Open terminal or command prompt and run:

node -v

npm -v

If versions appear, Node.js and npm are installed successfully.

## 4. Install VS Code
Steps:
1. Go to https://code.visualstudio.com
2. Download VS Code.
3. Install using the installer.

Recommended Extensions:
- ES7+ React Snippets
- Prettier
- ESLint

## 5. Create First JavaScript File
Steps:
1. Create folder 'javascript-practice'.
2. Create file 'app.js'.
3. Write code:

console.log("Hello JavaScript");

## 6. Run JavaScript File
Open terminal in the folder and run:

node app.js

Output will display in terminal.

## 7. Browser JavaScript
Create file index.html:

<script>
console.log("Hello from browser JS");
</script>

Open file in browser and check console.
