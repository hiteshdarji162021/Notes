### What it does
Performs a **right-click** on an element to open a context menu.

### Code
```ts

console.log(allOptions);
```

### Output
```text
[ 'Copy', 'Cut', 'Paste', 'Delete', 'Quit' ]
```

### Use case
- Context menus (IDE tools, admin panels)
- File explorers (right-click options)
- Desktop-like web apps

## 5ï¸âƒ£ Keyboard Actions & Tab Sequence

### What it does
Simulates **real keyboard navigation** using Tab, Enter, Space, typing.

### Code
```ts
await page.goto('https://naveenautomationlabs.com/opencart/index.php?route=account/register');

await page.locator('#input-firstname').focus();
await page.locator('#input-firstname').pressSequentially('Testing', { delay: 200 });

await page.keyboard.press('Tab');
await page.keyboard.type('Automation', { delay: 200 });

await page.keyboard.press('Tab');
await page.keyboard.type(`user${Date.now()}@mail.com`, { delay: 200 });

await page.keyboard.press('Tab');
await page.keyboard.type('9090909090', { delay: 200 });

await page.keyboard.press('Tab');
await page.keyboard.type('Automation@123', { delay: 200 });

await page.keyboard.press('Tab');
await page.keyboard.type('Automation@123', { delay: 200 });

await page.keyboard.press('Tab');
await page.keyboard.press('Space'); // checkbox

await page.keyboard.press('Tab');
await page.keyboard.press('Enter'); // submit
```

### Output
```text
Form filled and submitted using keyboard only
```

### Use case
- Accessibility testing (a11y)
- Keyboard-only user flows
- Forms without stable locators

## 6ï¸âƒ£ Real-World Data Validation Example â€“ Find Highest Score (Better Run)

This example shows how to **extract multiple values from a web table, process them in code, and derive meaningful business data** (highest score).

---

### Scenario
We want to find the **highest run scored** by a team from a live cricket scorecard.

---

### Code (Optimized & Clean)
```ts
import { Browser, chromium, Page } from "@playwright/test";

let page: Page;

(async () => {
  const browser: Browser = await chromium.launch({
    headless: false,
    channel: "chrome",
  });

  page = await browser.newPage();
  await page.goto('https://www.espncricinfo.com/series/icc-women-s-world-cup-2025-26-1478193/new-zealand-women-vs-south-africa-women-7th-match-1490419/full-scorecard');

  // Get all score values from first batting table
  const allScores: string[] = await page
    .locator('(//table[contains(@class,"ci-scorecard-table")])[1]//strong')
    .allInnerTexts();

  console.log('Raw Scores:', allScores);

  // Convert to numbers
  const numericScores = allScores
    .map(score => Number(score))
    .filter(score => !isNaN(score));

  console.log('Numeric Scores:', numericScores);

  // Sort and get highest score
  numericScores.sort((a, b) => a - b);
  const highestScore = numericScores[numericScores.length - 1];

  console.log('Highest Score:', highestScore);
})();
```

---

### Output
```text
Raw Scores: [ '15', '22', '45', '78', '3' ]
Numeric Scores: [ 15, 22, 45, 78, 3 ]
Highest Score: 78
```
