# What covered in notes

## Why learn?

- Multiple Elements with allInnerTexts() ‚Äì To quickly read and validate text from many elements without interacting with them.
- Multiple Elements with all() ‚Äì To loop over elements and perform real actions like click, check, or get attributes.
- filter() and hasText ‚Äì To safely narrow down the exact element you want and interact with it without writing manual loops.


## 1Ô∏è‚É£ `allInnerTexts()` + for-loop (Read all text & act)

### Code
```ts
let allHeaders: string[] = await page.locator('a.list-group-item').allInnerTexts();
console.log(allHeaders);

for (let e of allHeaders) {
  console.log(e);
  if (e === 'Forgotten Password') {
    await page.getByText(e).first().click();
    break;
  }
}
```

### Output
```text
[Login, Register, Forgotten Password, My Account]
Login
Register
Forgotten Password
```

### Use case
- Menu validation
- Clicking a link dynamically based on text

### Best practice
- Break loop once target is found
- Prefer `getByText()` for final action

## 3Ô∏è‚É£ `all()` with Locator array (Element-level loop)

### Code
```ts
let footerLinks: Locator[] = await page.locator('footer a').all();
console.log(footerLinks.length);

for (let link of footerLinks) {
  console.log(await link.innerText());
}
```

### Output
```text
10
About Us
Contact Us
Privacy Policy
```

### Use case
- Validate visibility or text of multiple elements
- Perform actions on each element

### Best practice
- Use `all()` only when you need **element-level actions**

## 4Ô∏è‚É£ Collect attributes using loop (`all()` + getAttribute)

### Code
```ts
let images = await page.locator('//img[@src]').all();
let altTexts: any[] = [];

for (let img of images) {
  altTexts.push(await img.getAttribute('alt'));
}
console.log(altTexts);
```

### Output
```text
['Amazon', 'Deals', null, 'Cart']
```

### Use case
- Accessibility testing
- Image validation

## 6Ô∏è‚É£ Search & select using `filter()` (SMART WAY)

### Code
```ts
let suggestions = page.locator(`ul li div[role='option']`);
await suggestions.filter({ hasText: 'python', visible: true }).click();
```

### Output
- "python" suggestion is selected

### Use case
- Auto-suggest search
- Dropdown filtering

### Best practice
- Prefer `filter()` over manual loops

## ‚úÖ Best Practices (MANDATORY)

- Use `allInnerTexts()` for **fast text reads**
- Use `all()` when interaction is needed
- Prefer `filter()` for search & matching
- Avoid nested loops
- Avoid XPath loops when possible

---
